# Beget's Blog

Welcome to Beget's Blog, a platform where you can find a variety of articles and tutorials on topics such as web development, software engineering, and programming languages.

## Features
- **Home Page**: Browse through recent posts and comments.
- **Admin Panel**: Manage posts, users, and comments.
- **Articles**: Read detailed articles on various topics like React and Material UI.

## Installation
To get started with Beget's Blog, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/behabtuG/My-Blog.git
   cd begets-blog
2. **Install Dependencies**:
   npm install
3. **Set up the database**:
   Ensure you have MongoDB installed and running.
   Update the database connection string in the .env file.

## Environment Variables
To run this project, you will need to add the following environment variables to your .env file:
## for backend
PORT=5000
MONGO_URI=Your_DB_Connection 
JWT_SECRET=your_jwt_secret
## for Frontend
VITE_FIREBASE_API_KEY = "your_firebase_api_key"
PORT = 5173
BACKEND_PUBLIC_API_URL=http://localhost:5000/
# Usage

    Start the server:
    npm start
   
    Access the blog:
    Open your browser and go to http://localhost:5173.

    Use the admin panel to manage content by navigating to http://localhost:5173/dashboard?tab=dash.
